# Bari → Matera

## Transport
- Bus Bari – Matera (a organiser)

## A voir
- Les Sassi di Matera (UNESCO) — deux quartiers : Sasso Barisano (plus touristique) et Sasso Caveoso (plus brut, authentique). Se perdre dans les ruelles.
- Casa Grotta — maison-grotte reconstituée, montre la vie dans les Sassi jusque dans les années 1950. ~3 EUR, rapide et fascinant.
- Belvedere Murgia Timone — de l'autre côté du canyon, vue spectaculaire sur les Sassi. Idéal pour le coucher de soleil.
- Églises rupestres — Santa Maria de Idris (fresques byzantines), Santa Lucia alle Malve, San Pietro Barisano (plus grande église souterraine).
- Cathédrale de Matera — au sommet de la Civita, entre les deux Sassi. Style roman, belle vue du parvis.
- Palombaro Lungo — citerne souterraine géante sous la Piazza Vittorio Veneto. Visite guidée ~30 min.
